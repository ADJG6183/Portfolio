Portfolio Update Instructions

Follow these steps to update your portfolio with the new design:

Option 1: Replace Files (Recommended)

  1. Rename index-new.html to index.html (replace the existing file)
  2. Rename styles-new.css to styles.css (replace the existing file)
  3. Rename script-new.js to script.js (replace the existing file)
  4. Rename styles-cursor-new.css to styles-cursor.css (replace the existing file)

Option 2: Keep Both Versions

If you want to keep both versions before deciding which one to use:

  1. Open index-new.html in your browser to see the new design
  2. Compare with the original design by opening index.html
  3. Make your decision and then follow Option 1 if you want to use the new design

Terminal Commands (Option 1)

You can use these commands in your terminal to replace the files:

cd /Users/aaron_green/Desktop/portfolio
mv index-new.html index.html
mv styles-new.css styles.css
mv script-new.js script.js
mv styles-cursor-new.css styles-cursor.css
